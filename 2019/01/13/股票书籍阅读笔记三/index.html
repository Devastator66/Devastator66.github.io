股票书籍阅读笔记三

发表于 2019-01-13 分类于 Economy 阅读次数：

看盘细节

股市中有两种人，一部分是猜股价走势，而另一部分是做股价走势的。

大资金操盘手一般会实时关注盘中每一笔交易。

操作指标的三个方法：

  1. 即将见顶前，尽量把股价打高。在还没有大压盘的情况下打高股价，这样会形成一个没有多大成交量的最高价。
  2. 在第一天回落时，加大出货力度，打出较低的低点，并形成宽幅震荡的态势。因为卖盘不大，抢反弹的人很多，主力趁机出货。
  3. 隔天奋力上推。由于前一天跌幅较大，一般除非第二天大盘大跌，否则市场会有一批买单进场。

如果主力不再对倒，成交量会迅速回落，股价进入调整状态。如果成交量大幅萎缩，反而时市场自然表现。

主力特征

  1. 开设多个账户；
  2. 进行一些虚拟（会撤单）的买卖申报；
  3. 具备大笔对敲交易的能力。

主力在运作项目的过程中必须得到市场的配合。他们在建仓时希望市场抛盘出来，会挂出大卖单压制股价。进入拉升阶段又会做出双底，头肩底，箱形突破等上升形态。

业绩得到提升，股价会涨，但肯定有人提前知道这些消息。当市场都知道结果的时候，股价通常已经做出了反应。被主力看中的股票，基本面上一定会有一定程度的保证。

要发现主力建仓是比较困难的。

在拉升阶段希望市场跟进的真正意图是提高市场筹码的换手率，从而提高市场持有筹码成本的目的。

一只股票高台跳水，在相对底部通常会有抢反弹的。此时，庄家还会继续出货，直到没有什么人买了，庄家会稳步吃货，此时成交量不会放大，且价格不会过涨。

底部放巨量，但股价不怎么涨，是庄家在出货。

开盘价

  1. 平价，大成交量。原因是庄家对倒。
  2. 跳空高开、低开。
    没有普通人在开盘前，以过高或过低的价格竞价，只有主力才会这么做，为了输送利益，不让散户参与。尤其是巨量低开盘，完全是把利益喂给某个利益集团。当哪个集团想退出时，就会出现巨量高开盘。如果接下来的交易恢复正常波动，就证明了前面的猜想。所以策略就是低开买入，高开卖出。
    高开就是想吸引买盘，主力肯定不在建仓期，已经不需要更多筹码，所以是否跟进取决于后续观察。如果之前一直亮红可以追涨；如果之前已经下跌，这是一波回弹，那么跌的可能性更大。
  3. 开盘是大压盘后跟一堆小接盘。
    一般出现在建仓期，主力实力有限，通常是短线主力。一堆小抛盘后跟大接盘，通常是主力的护盘行为。

放大量都算是非正常情况。

如果大盘上冲，开盘股价上冲是正常的。如果大盘平静，甚至下跌，股价还上涨就不正常了。原因有两个：公司利好即将发布，或是主力参与。主力一般最了解市场的习惯性思维。

除非牛市期间或者个股处于主力运作期间，否则一般的股票成交量都比较小。主力通过对倒将放量的力度尽量加大。此时，主力是希望散户跟进的。原因有：主力不愿增加筹码，此时不是主力建仓期或拉升初期。

当股价远离主力成本区，主力才希望买单真正介入。

主力将股价打高，可能是为了打击抛盘，但又不愿意增加筹码，在大家观望时快速打高股价。只要主力放手，股价立刻就会回落。要让股价走势强于大盘而不增加筹码是一名职业操盘手的最高境界。

收盘价

  1. 上下平衡，这是比较自然的交易状况；
  2. 高空形态，上档的卖出价远离收盘价，而买进则贴近或等于收盘价，就是说卖盘很空虚。如果交易清淡，那么就是正常的，盘中无主力，或者主力不愿意做收盘价。如果大盘快速下杀，而股票出现这种高空，那么说明有资金在吸筹，不断将上档的抛盘打掉；
  3. 低空形态，下档的买入价远离收盘价，而卖出价则贴近或等于收盘价，就是说买盘很空虚。如果当日成交稀少，那么低空形态是正常的。如果尾盘上涨，正常情况下会有散弹挂在下面，不可能空虚，所以有主力在运作收盘价。

主力有两种方法做收盘价：

  1. 最后一两分钟打高股价，条件是上档压盘很小；
  2. 借助于大盘尾市的稍稍好转而个股上档的压盘不大，主力可以在尾市的一段时间内尽可能地将上档的单子打掉直到收盘。

无论哪一种方法，都说明盘中有主力而且主力不希望股价继续下跌，属于明显的护盘动作。盘中的主力绝不是新的建仓主力。

证劵市场的活力在于交易，一个市场成功与否的重要标志之一就是市场的换手率。真正吸引市场资金介入的首要因素收盘价，而不是成交量。

一家企业要想引起大资金的注意，首要是基本面出现变化。

底部本来是没有量的，所以一定是受到非市场性因素的干扰。除非有基本面上的变化或大盘暴跌，否则不会出现底部放量。

  1. 底部放量上涨。此时主力希望市场跟单。说明主力早已完成建仓，甚至被深套。我们要观察股价上涨时的成交量，如果量不是太大，可以提高一些期望；如果放量巨大，那么可能就是主力快速出货。
  2. 底部放量后下跌。一般就是主力弃庄，以收回资金。主力更理性的做法是找另一个主力，约定在某个更低的价位交换筹码。
  3. 底部放量后盘整。可能1.主力护盘；2.分仓；3.换主力，盘中会有大单换手。

底部永远是市场关系的重点，真正的底部是没有成交量的。底部不是量堆出来的，是自己走出来的。

无量涨停

  1. 下跌放量底部有量而涨停无量。价格跌落时放量，说明有主力出逃。底部又放量是因为吸引了大量的短线客，如果放大量，则可能有短庄接手，此时因为资金充沛，且成本位置明显，所以希望在抛盘出来前，将股价打的高一点。
  2. 下跌无量底部无量而涨停无量。涨停无量说明有主力参与，因为无量说明主力没有减仓意愿，看涨。

巨量涨停，主力的成本不在前面的底部（被套的主力），巨量是对倒出来的。

封不住的涨停，回落幅度不能过大，否则就很难再封到涨停。断断续续的涨停可以理解为主力想收集筹码，拉高建仓。这种现象一般出现在股价的相对低位。

盘中巨量，如果之前股价过高或基本面不好就不要买。

没有什么成交量的股票，如果有一卖单，会将股价打的很低。我们应该注意力集中在交投活跃的股票上，这种股票有主力参与的可能性相对大一些。

在刚刚离开底部就急急忙忙做出巨量，表明是老主力，甚至是被套的主力在运作。我们不应该将目标定的过高。

主力一般都是将股价运作到个股的基本面有一个明朗的结果为止。这就是判断主力是否出货的一个重要依据。因为在消息没有明朗之前很难做到全身而退。

如果主力减仓，那么在盘中绝不会在上方压出很大的抛盘，而是下方有接盘就会砸出去。

通过股价的回调震出获利筹码，关键是要让新的买盘进场承接，最终目的是提高市场的平均持筹成本。

上档有大压盘，下挡有几个大接盘。主力不想让股价涨高，且在下面承接抛盘。

阻截式大单，大单的出现才说明市场的活跃，因此要把更多的精力花在大单的研究上。在接盘中有一档大接单顶在那里，阻止了股票下跌。这说明想抛的大单不多，也可能市场已经没有多少大单了。主力可能锁住了大单，导致没有大单出逃，主力的仓位已经相当重了。主力此刻已不想再接任何单子，而是想出货。

小单的真实感，如果5个接抛盘挂单都非常小，那就是市场的真实意图，应该是没有主力运作。挂单的放大则意味着主力的参与。如果在主力运作过程中出现了大卖单，最多两次，主力就会放弃运作。

主力运作的意图有两个，一个是要告诉市场股价即将怎么样，一个是通过告诉市场一些东西来达到主力自己的真正目的。

巧妙撤单，在上方5档压单，然后同时消失。目的可能是探一下底部，或让非主力买单吸货。

下空走势，上档都是大抛单，下档都是小买单，可是价格还在上涨。说明买盘直接打掉卖单，而不是通过挂单。主力通过对倒的方式在拉升股价，因为不愿意大量增加筹码，所以下面不敢接大买单，而且一旦下面出现稍大一些的接盘立马被主力喂掉。说明主力想要减掉一些仓位，还有就是目前的价位获利盘不多。

间断性交易，通过分时图的成交量发现，大单的交易是断断续续的，大盘大涨，但股票下跌，主力想造量吸引市场买单，要减仓。既然主力愿意在低位减仓，即使以后大盘再好，其股价上升力度也会打折扣。交易的连续性是万万不可忽视的。

闪烁的买盘，接盘不停的挂单撤单，说明有主力存在。主力撤掉单子，让后面的单子成交，然后再重新挂上自己的单子，循环往复。主力可能是想吸引买单，威吓卖单。主力可能被套了。

尾市拉抬的用意，需要结合全天的表现进行分析。

  1. 该股全天成交平淡，大单很少。说明市场已经将该股遗忘，结论是主力护盘。
  2. 盘中股价在一个特定的较低的价位附近出现过一些较大的成交量。由于股价没有继续下跌，说明主力与他人有默契。主力做尾盘是为了护盘。
  3. 盘中出现主动性往下打压的单子。答案还是护盘，为了在第二天以相对高的价位出货。主力因为知道外面筹码不多，所以敢狠砸以降低股价，吸引买单。出货者是绝对不会再花大资金做收盘价。

成交清淡的背后，越成熟的市场，越可能清淡，一天可能不会成交一手，如果中间出现了大单，要引起注意。

基金的进出，随着市场的发展，基金是一股重要的博弈力量。基金在发行的时候就已经将自己的投资方向做了一个大致的确定。各家操盘方式类似，一般有研发、决策和交易三个过程。基金操作员会尽可能短的时间完成任务。如果只是一张机构大买单，那么一旦这张单子完成，股价依然会跟随大盘起落，当天过大的涨幅会得到修正。

私募基金一般挂投资公司的名分，其组织结构简单，主要决策往往只有一二人。如果公司决定买进，一般只是大约确定一个价格范围，没有完成单子也没有关系。更多情况下，操盘手完成单子的质量都是优于原先计划的。

基金购买的股票，一般基本面没什么问题。基金最容易出错的是大势的研判。一旦大盘下跌，所有的股价都会重新定位，因此即使股票的公司基本面没问题，也顶不住股价回调。基金不是主力，很少主动护盘。

股价启动前的征兆，如果被套，不要卖在地板上，应该先分析一下是否有反弹的可能：

  1. 较大的卖单被打掉，没有什么抛盘的股票，主力才会拉升。
  2. 盘中出现一些非市场化的大单子，只是挂着，离成交的位置较远，所以不会实际成交，目的是告诉市场，有人在注意这家股票。
  3. 盘中出现脉冲式上冲行情。突然脱离大盘走势上冲，然后快速回落到原来的位置，没有放大成交量。这是主力的试盘，看看上面的抛单如何，或者提前打掉一些抛单。
  4. 盘中出现压迫式下探走势，但尾市往往回稳。主力诱空，希望买到更多的低价筹码。

楼板的消失，说明有主力存在，实际市场上的买单很少，主力想要试试卖方的力量。

本身的投资决策指短线的一次投机决策，这也需要考虑一下个股的基本面和大盘背景。经典图形分析理论的主要缺陷在于完全脱离了大盘和主力这两个最基本的因素。

分析方法：
一、在近期日K线图上寻找成交量急剧放大的地方。如果量放的不是很大，而且放量的日子正好是大盘同步上涨的日子，那么就认为没有主力。这里指的是脉冲式放量而不是连续放量。

  1. 长期下跌后V形回升，数天后涨停开盘，放巨量，当天收阴，随后股价日渐回落。主力做量吸引买单，目的是为了减仓。说明主力已经满仓，而且应该还有一定的资金。
  2. 长期下跌，底部放量但股价波动不大。说明盘中有默契的换手。该股可能有有一些短线机会，除非大盘不配合。
  3. 长期下跌，出现回升，走势强于大盘，每一天上升都伴随巨量。说明主力随着股价回升连续大量对倒，想要吸引买单。由于在长期低位，主力一定被套，在低位减仓，说明其仓位过重且资金匮乏。这种走势，前景不妙。

二、对比股价与大盘的相对强度。我们绝不能以独立的方式进行个股的分析。个股必须强于大盘，才值得分析。相对强度 = 股价 / 大盘指数 * 100。如果这根指标上行，股价就强于大盘。所谓走强是相对的，个股可能是下跌的，只是幅度小于大盘而已。上涨比下跌难得多。在大盘下跌的时候，个股跌幅明显小于大盘，说明受到了外力的干扰。

相对强度指标的较强状态

  1. 每次大盘回落的时候股价的回落都比大盘少。属于主力护盘，而且绝对是老主力。要是新主力，会顺势砸盘。
  2. 大盘回落的时候股价小幅回升。要么是新主力建仓，要么是机构的一次性买单，需要借助于分时图确认。
  3. 大盘上升的过程中股价比大盘涨的更多。基本上判定为市场上的大买单，不是主力运作，因为不会护盘，所以还会回落。
  4. 股价远远走强于大盘，个股独立于大盘走出连续上冲行情。这是主力的拉升阶段，但还要确认新老主力的差异。

如果相对强度走弱，那么就没必要关注这支股票了。

三、一般市场认同“价涨量增后市看好”这个理念，主力在拉升股价的过程中会操纵成交量。我们更关心大盘下跌的图形，于此对应个股有三种可能：上涨、盘整、下跌。如果股价逆大盘而大幅上涨，表明主力已经进入拉升阶段，这不是我们感兴趣的范畴。主要分析大盘下跌而股价盘整的情况：

  1. 持筹者非常看好该股的前景而不愿抛售。看看是不是市场热点，如果不是，要看成交量是否很小；
  2. 已经在长期下跌的底部而出现惜售。关键也在于成交量，惜售的结果一定是没有量，跌无可跌。短线操作不会关注这种股票；
  3. 存在某一大买单在目前价位买入。出现的连续放量，可能来自主力或机构。要确定是否是主力，需要看盘中抛接盘细节。更方便的方法是，通过查看相对强度上升波的次数。如果在几个月内“相对强度”有过3次甚至更多的上冲次数，那么就可以确认主力运作；
  4. 主力在护盘或者建仓。相对强度多次上冲，且买单量比较大。不是新主力，因为新主力会顺势压盘。可能是老主力加仓、阶段性主力或者大机构联合买入，无论哪种都有短线机会。

如果只是脉冲式大量，说明主力被动性护盘，这种主力由于已经满仓或者已经没有多余的资金，未来股价的走势不会很强。这种放量很少会连续超过2天以上，而且因为是对倒，所以量放的特别大，与这里分析的连续放量情况明显不同。脉冲式放量的前提往往是大盘的上涨日。

一些佐证

  1. 没有脉冲式放量现象；
  2. 相对强度有3次以上的上冲；
  3. 相对强度上冲过程中，大盘没有上涨；
  4. 相对强度上冲过程中，放量但不是巨量。

一些容易忽视的细节

  1. 股价拉出长阳线或长阴线。此时要对比一下大盘。大阳线没有明显的意义。因为大盘如果下跌，说明主力不在建仓区域；大盘上涨，个股阳线也是正常的。大阴线更有意义。如果是新主力在建仓，会顺势下压股价以买入更低价筹码。如果是主力阶段性增仓，也可能因为持仓较多而大盘没有起色时出掉一些筹码。这样就没有动摇前期的分析。
  2. 股价波动幅度。波动幅度过大的个股表明主力在其中挣扎，后面难以走出强于大盘的可操作性行情。我们希望得到波动幅度比较小的个股，这表明有新主力在建仓或者老主力在进行阶段性增仓。
  3. 较长的上影线或者下影线。如果当天在最高价或最低价附近成交的量非常小，那么可以认定只是偶然因素。由于历史分时图已经看不到，因此可以通过计算当天的成交均价做一个大致判断。

投资中最简单的事

  3. 商业模式，行业属性研究透，要对行业有洞察力；
  4. 逆向思维和独立思考。

散户有天然的信息劣势，所以适合在消费品、医药和零售业投资。

始终用超高仓位进行集中的逆向投资，用好的价格买入好的企业，这本身就是最好的风险控制。

投资最重要的事

  1. 最重要的不是趋势而是周期；
  2. 最重要的不是追求高风险高收益而是追求低风险高收益；
  3. 最重要的巨大风险不发生于人人恐惧时而发生于人人都觉得风险很小时；
  4. 最重要的不是主动寻找机会而是耐心等机会上门；
  5. 最重要的是认识到短期业绩靠运气而长期业绩靠技术；
  6. 最重要的投资决策不是以价格为本而是以价值为本；
  7. 投资更像是一门艺术而非科学没有放之四海而皆准的法则；
  8. 价值投资者的目标是得出股票当前的内在价值并在价格低于当前价值时买进成长型投资者的目标则是寻找未来将迅速增值的股票；
  9. 估计正确却不坚定地持有用处不大；
  10. 成功的投资除了买好的更在于买得好。
  11. 投资是一场人气竞赛在人气最旺的时候买进是最危险的；
  12. 最安全、获利潜力最大的投资是在没人喜欢的时候买进；
  13. 使用杠杆不会使投资变得更好也不会提高获胜概率，它只是把可能实现的收益或亏损扩大化，这个方法还引入了全盘覆灭的风险；
  14. 在以低于价值买进股票的时候高收益、低风险是可以同时实现的；
  15. 万物皆有周期，树木不会长到天上；
  16. 投资市场遵循钟摆式摆动；
  17. 随着定价过高的股票走势更好，或者定价过低的股票持续下跌，周期的做法变得更加简单：卖掉前者，买进后者；
  18. 最好的机会通常是在大多数人不愿意做的事情中发掘出来的；
  19. 买进的绝佳机会出现在资产持有者被迫卖出的时候，在经济危机中，这样的人比比皆是；
  20. 我们有两类预言家：无知的和不知道自己无知的。

如果一项资产鲜为人知，追随者不多，那么它可能就不会得到有效的定价；如果一项资产是争议性的或未被社会认同，那么它可能就不会有效定价。

市场价格往往是错误的，因为获得信息的途径以及对信息的分析不完善，所以市场价格往往远高于或低于内在价值。找出未被完全有效定价的市场或资产，如果市场是有效的，那么就赚不到钱。内在价值是低卖高卖的基本出发点。

价值投资强调的是有形因素，重视资产和现金流，首要目标是确定公司的当前价值，并在价格足够低时买进公司证券。

成长型投资者赌的是未来可能实现，也可能无法实现的公司业绩，而价值投资主要建立在分析公司当前价值的基础上。

成长的概念是被纳入到价值计算当中的。

  1. 准确判断企业的价值；
  2. 坚定的持有。

估价正确却不坚定持有，用处不大；股价错误却坚定持有，后果更糟。这句话表明恰到好处是多么困难。

对利润的追求必须建立在更加有形的东西之上，最佳选择就是基本面所反映出来的内在价值。

当价值投资者买进定价过低的资产，不停的向下摊平并且分析正确的时候，就能获得最大收益。

建立基本面-价值-价格之间的健康关系是成功投资的核心，必须以合理的价格买进，成为强制卖家是世界上最悲惨的事。

确定价值的关键是熟练的财务分析，而理解价格、价值及其前景的关键，则主要依赖对其他投资者思维的洞察。

基本面价值只是决定证券价格的因素之一，你还要设法让心理和技术为你所用。

某种东西价格上涨时，人们的喜爱程度本应下降。从真实价值出发的投资方法最可靠。问题是价值的增长性难以准确预测。

风险意味着可能发生的事件多于确定发生的事件。靠更高风险的投资来获得更高的收益是绝不可能的。

价格过高意味着低收益、高风险。

概率与结果之间存在巨大的差异，可能的事情没有发生，向来如此。
