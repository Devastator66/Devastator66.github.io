数据可视化

发表于 2020-11-01 分类于 Python

管理工作（非领导）很难界定绩效，在当前中国职场远不如技术岗、销售岗有前途，在某些方面甚至还不如生产岗！所以，作为管理人员不能消极地寄托于领导具有一双慧眼并且具有较高的悟性，应该将自己的工作以高水平的方式展现出来。
