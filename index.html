Python3基础语法

发表于 2019-05-05 分类于 Python

基本特性

  1. 弱类型语言。任何数据都是对象。
  2. 代码区分大小写，变量可由下划线、数字、字母组成。
  3. 不使用花括号标识代码块，而是采用缩进的形式。
  4. 不使用任何符号标识每行代码的结束（分号可用来分隔一行中的多个表达式）。
  5. 引入了编码-探索开发模式。
  6. 代码库特别丰富，支持各种应用领域。
